# TezDel Brain

TezDel Brain is the autonomous AI operating system dashboard for TezDel.

It encodes the master operating prompt as structured product data: the CEO command center, AI departments, daily execution loop, collaboration protocol, operating doctrine, KPIs, and long-term expansion goals.

## Run Locally

```bash
npm install
npm run dev
```

The local dev server runs on `http://localhost:5175`.

## Build

```bash
npm run build
```
